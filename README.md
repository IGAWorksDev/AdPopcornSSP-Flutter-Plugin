# adpopcornssp

AdPopcornSSP plugin project.

## Current AdPopcornSSP SDK version

- Android :  3.4.1
- iOS : 2.6.5

## Install the Plugin

Add dependencies on pubspec.yaml

```
dependencies:
  adpopcornssp: ^1.0.0
```
